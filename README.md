# Lista completa de todas las insignias y logros del perfil de GitHub

#### Mostrar los logros en tu perfil es completamente opcional; de forma predeterminada, pueden ser vistos por cualquier persona que vea su perfil público.

Puede optar por no mostrar los logros en su perfil yendo a su [configuración de perfil] (https://github.com/settings).
